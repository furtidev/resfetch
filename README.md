## Installation

### Cargo
```bash
$ cargo install resfetch
```

### Homebrew
```bash
# tapping the formula
$ brew tap hitblast/resfetch

# installing it
$ brew install resfetch
```

### Docker
```bash
# building image
$ docker build -t resfetch .

# running
$ docker run resfetch
```
